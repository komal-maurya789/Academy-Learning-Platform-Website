🎓 Academy Website (WordPress Project)

📌 Project Overview
This project is a modern and responsive Academy Website built using WordPress. It is designed to provide online learning services, showcase courses, and connect students with industry-ready skills.
The website includes multiple sections like courses, services, team members, testimonials, and contact forms — all designed with a clean and user-friendly UI.

🚀 Features
✅ Fully Responsive Design
✅ Built with WordPress (CMS)
✅ Modern UI/UX Design
✅ Course Showcase Section
✅ Digital Services Section
✅ Team Members Section
✅ Testimonials Section
✅ Contact Form Integration
✅ Call-to-Action Sections
✅ SEO-Friendly Structure
